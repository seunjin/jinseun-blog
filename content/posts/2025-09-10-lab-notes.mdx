# Panda 패턴 조합하기

Panda CSS의 `stack`, `hstack`, `grid` 패턴을 활용하면 레이아웃을 선언적으로 표현할 수 있습니다. 특히 다단 디스플레이에서 gap과 wrap을 조합해 다양한 레이아웃을 손쉽게 구현할 수 있었어요.

<Figure
  src="https://images.unsplash.it/1200/675?grid"
  alt="Panda CSS patterns"
  caption="패턴 조합으로 만든 대시보드 배치 예시"
  width={1200}
  height={675}
 />

```tsx
import { hstack } from "@/styled-system/patterns";

const actions = hstack({
  gap: "4",
  flexWrap: "wrap",
  justify: "space-between",
});
```

<Callout type="tip" title="레이아웃 팁">
패턴을 사용할 때는 gap과 wrap 조합만으로도 대부분의 반응형 UI를 커버할 수 있습니다. 기본 토큰을 사용하면 다크 모드에서도 색상 대비가 자동으로 맞춰집니다.
</Callout>

## 실험에서 얻은 인사이트

- 토큰 기반 spacing과 typography만으로도 일관성을 유지하기 쉬웠습니다.
- 컴포넌트 레벨에서 variant를 정의해두면, 관리자 페이지에서도 재사용이 수월합니다.
